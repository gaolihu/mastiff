### 键盘
&#8629; 表示回车键
2 c 1 &#8629; 表示先按2,再按c 再按1 再按回车

|2 c 1 &#8629;   | 网络  |  获取工作模式 |   |

|2 c 4 &#8629;   | 网络  |获取DNS   |   |

|2 c 9 &#8629;   | 网络  |获取扫描列表   |   |
|2 c 10 &#8629;   | 网络  | 重连  |   |
|2 c 11 &#8629;   | 网络  | 断开连接  |   |
|2 c 12 &#8629;   | 网络  | 检查是否已经联网  |   |
|2 c 13 &#8629;   | 网络  | 网络测速  |   |
|2 c 14 &#8629;   | 网络  | 连接WIFI  | 需要传入wifi名和密码   |
